Exercício 1

Você aprendeu que o JavaScript pode ser usado diretamente no navegador e decidiu praticar usando apenas a ferramenta de console. Seu objetivo é escrever a mensagem "Estou estudando JavaScript!" no console.

Exercício 2

Você foi desafiado a implementar uma função que subtrai dois números. Para isso, você pode usar a ferramenta do Console do navegador para te ajudar a implementar essa lógica. Uma dica é você criar duas variáveis a e b para armazenar os valores e exibir no console a soma dos dois números.

Exercício 3

No exercício 1, você escreveu a mensagem "Estou estudando JavaScript!" no console do navegador. Agora, a ideia é dar um salto e evoluir essa mensagem. Seu objetivo é personalizar a mensagem e permitir que a pessoa usuária defina qual tecnologia ela está aprendendo. Uma dica é criar uma variável que armazene o nome da tecnologia escolhida e interpolar a variável com a mensagem no console do navegador, para tornar o texto dinâmico.
